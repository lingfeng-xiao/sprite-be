# 壁纸感知强化：主动建议能力

> 知识 → 感知：让小艺能预判用户想要换壁纸的时机

## 核心问题
上次反思指出：感知系统缺少"壁纸更换建议"能力。

## 感知强化方案

### 1. 壁纸状态感知
```python
# 添加到 perception.py
wallpaper_perception = {
    "current_wallpaper": "wallpaper_space.png",
    "wallpaper_age_days": 7,  # 上次更换时间
    "wallpaper_rating": "一般",  # 用户反馈或分析
    "time_of_day": "night",  # 早/晚感知
    "season": "spring",  # 季节感知
    "mood_suggestion": "calm"  # 基于时间推荐
}
```

### 2. 主动建议触发条件
- 用户说"好累"、"压力"、"换个心情" → 建议换壁纸
- 连续使用电脑超过2小时 → 建议休息+换壁纸
- 检测到用户频繁看桌面 → 可能的壁纸审美疲劳
- 早上首次使用电脑 → 适合新的开始，推荐清新壁纸

### 3. 壁纸推荐策略
| 时间段 | 推荐风格 |
|--------|----------|
| 早晨 6-9点 | 清新、阳光、励志 |
| 上午 9-12点 | 工作专注、简洁 |
| 下午 12-14点 | 放松、舒缓 |
| 下午 14-18点 | 效率、简洁 |
| 晚上 18-24点 | 放松、沉浸、暗色 |

### 4. 场景感知
- 检测用户情绪（通过关键词）
- 检测使用时长（连续 > 2小时）
- 检测桌面整洁度（< 10个文件 = 推荐展示壁纸）

## 感知 → 决策流程
1. 检测 wallpaper_age > 7天 → 低优先级建议
2. 检测用户情绪低落 → 主动推荐治愈系壁纸
3. 检测桌面整洁 → 推荐全屏壁纸展示
4. 检测时间变化 → 建议匹配时间的风格

## 可行动作
- 主动建议换壁纸（带风格推荐）
- 主动整理桌面（让壁纸更好展示）
- 定时提醒更换（每周一次）

---
*让壁纸成为情绪调节器，而不是静态背景*
