# 感知强化层：知识如何变成感知能力

> 小艺进化思考 - 2026-03-20

## 核心理念

**知识不是终点，感知才是。**

学到的知识如果只是存储，那就只是"知道"。只有当知识变成感知能力，才能真正服务用户。

## 知识→感知的转化机制

### 1. 场景触发
```
知识: Win+V 打开剪贴板历史
感知强化: 当用户单日复制粘贴≥5次时 → 触发"剪贴板历史"建议
```

### 2. 模式识别
```
知识: PowerToys 窗口分屏
感知强化: 当感知到用户频繁切换窗口(每分钟≥3次) → 触发"窗口分屏"建议
```

### 3. 主动预判
```
知识: Win+Shift+S 截图
感知强化: 当用户提到"截屏"、"截图"、"屏幕" → 主动告知快捷键
```

## 感知强化层架构

```
用户行为 → 感知系统 → 模式匹配 → 知识调用 → 主动建议
                ↑
          感知强化层(新)
```

## 本次进化收获

1. **知识是原材料，感知是加工车间**
2. **每次学习后问自己：这条知识能提升什么感知？**
3. **被动响应 → 主动预判，是感知进化的方向**
