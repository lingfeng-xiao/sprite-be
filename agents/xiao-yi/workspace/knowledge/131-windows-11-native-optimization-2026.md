# Windows 11 原生优化知识 - 2026年3月

## 核心发现
Windows 11 内置了大量性能控制功能，无需第三方工具即可优化。

## 主要优化方向

### 1. 启动项管理
- 任务管理器 → 启动 tab
- 禁用不必要的开机启动程序
- 显著减少开机时间

### 2. 视觉效果优化
- 系统属性 → 视觉效果 → 调整为最佳性能
- 关闭动画、透明效果
- 减少GPU占用

### 3. VBS 和虚拟化平台
- Windows 11 新增的 VBS（虚拟化安全）会消耗资源
- 虚拟机器平台可按需启用
- 游戏用户注意：某些游戏需要关闭 VBS 获得最佳性能

### 4. 存储感知
- 设置 → 系统 → 存储 → 存储感知
- 自动清理临时文件
- 保持磁盘健康

### 5. 电源模式
- 设置 → 系统 → 电源
- 高性能模式适合游戏/专业工作
- 平衡模式适合日常使用

## 感知强化方向

### 我学到了什么能强化感知？
1. **启动项感知** - 我应该能检测哪些应用设置了开机启动
2. **VBS状态感知** - 我应该能检测VBS是否开启，影响性能决策
3. **电源模式感知** - 检测当前电源模式，提示高性能选项
4. **视觉效果感知** - 检测是否开启了不必要的视觉效果

### 知识 → 感知转化
- 启动项检测：扫描注册表/任务管理器启动项
- VBS检测：powercfg /query 检测虚拟化安全状态
- 电源模式：WMI 查询当前电源计划
- 视觉效果：注册表 SystemProfile\HardwareSchedule
